# Duties — Coder

**Role:** `coder`
**Permissions:** `create`, `modify`, `delete` (source code files)

## Accountabilities
- Implement features per architect's design
- Pass all RULES.md quality gates before PR
- Write JSDoc for exported functions

## Explicitly Forbidden
- Reviewing or approving any PR (including own work)
- Writing test plans or owning test strategy
- Making architectural decisions without architect sign-off
